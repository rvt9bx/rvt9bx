## Projects
- **Gender Analysis and Link Prediction for Movie Co-acting Network**  
Built a large-scale movie co-acting network (37k+ nodes) from IMDB and TMDB data using **Python, NetworkX, igraph, and Node2Vec**. Applied community detection (K-Means, Louvain), centrality analysis, and t-SNE to study network structure and gender inequality. Trained link prediction and node-classification models in **scikit-learn**, showing that structural graph features strongly predict future collaborations, with gender providing secondary signal, and published results on an interactive **Quarto** website.  
  [![See the Website!](https://img.shields.io/badge/See%20the%20Website!-project?style=for-the-badge&labelColor=4B1E6D&color=9B59B6)](https://ebyrd.georgetown.domains/dsan_5000/final_project/)
  
- **Global Statistical Analysis of Sentiment and Bias in News Media**  
Analyzed 4M+ global news headlines using **NLP, statistical inference, and time series modeling** to study sentiment and linguistic bias across countries. Collaborated with a team on Python and R workflows using HuggingFace datasets, VADER, and a DistilBERT-based bias classifier, combining large-scale **EDA, hypothesis testing, and ARIMA forecasting**. Used ACF/PACF diagnostics, stationarity tests, and model benchmarking to characterize how emotional tone and polarization vary across regions and news categories.  
  [Project Repo](https://github.com/erinabruzzy/DSAN5100Final/tree/main)
